## Git 초기화 및 푸시

```git
git init
git branch -m main
git remote add origin "~/~/~.git(경로)"
git add .
git commit -m "Comment"
git push -u origin main
```
<br/>

## 커밋 로그 보기

```
git log
```
<br/>
