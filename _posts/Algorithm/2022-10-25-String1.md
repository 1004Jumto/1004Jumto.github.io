## StoI

```cpp
    int stoi(string s){
        
        int res = 0;
        int len = s.length();

        for(int i = len-1, p = 1; i >= 0; p*=10, i--){
            int tmp = s[i] - 48;
            res += (tmp * p);
        }

        return res;
    }
```

## ToString, ItoS

```cpp
    string itos(int n){
        string res = "";
        
        while(n != 0){
            char tmp = (char)(n % 10) + '0';
            res = tmp + res;
            n /= 10;
        }

        return res;
    }
```
